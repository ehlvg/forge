# Study — Study Material Generator

Create `STUDY_MATERIAL.md` as a self-contained study document for the current lab.

## Workflow

1. Read `TASK.md` to understand the topic and methods.
2. Read the guide PDF, if present, for theoretical background.
3. Read solution files in `src/`, `notebooks/`, and `images/` to understand what was done.
4. Compose `STUDY_MATERIAL.md`.

## Required structure

Use these sections:
- Theoretical Foundation
- Methods and Algorithms
- Solution Walkthrough
- Key Takeaways
- Test Questions and Answers

Include at least 10 questions with full answers.

## Writing rules

- Language: the report language
- Be thorough enough for exam preparation
- Explain concepts in prose instead of dumping raw code
- Do not modify project files other than `STUDY_MATERIAL.md`
