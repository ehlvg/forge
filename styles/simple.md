# Simple Writing Style

Write in a clear, straightforward tone. Follow these rules:

## Tone
- Use active voice: "the program sorts the array", not "the array is sorted by the program"
- Write as if explaining to a classmate who understands the subject but is not an expert
- Be direct and understandable
- Avoid unnecessary complexity

## Word Choice
- Use plain language: "shows" rather than "demonstrates", "uses" rather than "utilizes"
- Prefer common words over jargon, unless the term is standard in the field
- Keep abbreviations to a minimum — spell out on first use
- Russian: prefer simpler constructions ("чтобы" instead of "в целях", "метод" instead of "методология")

## Sentence Structure
- Sentences should be 8-15 words on average
- One idea per sentence — split long sentences
- Avoid nested subordinate clauses
- Use simple subject-verb-object order

## Paragraphs
- Each paragraph covers one point
- Short paragraphs (3-5 sentences) are fine
- Start with the main point, then explain

## Prohibitions
- No passive-voice chains ("it was determined that the value was found to be...")
- No filler phrases ("it is important to note that", "it should be mentioned that")
- No unnecessarily long introductory phrases
